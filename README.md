# 🌦️ Weather & Air Quality Dashboard (Power BI)

## 📊 Overview

This project is a real-time **Weather and AQI Dashboard** built using Power BI and WeatherAPI.

---

## 🚀 Features

* Live Weather Data (Temperature, Humidity, Wind Speed)
* Air Quality Index (AQI)
* AQI Status & Suggestions
* 7-Day Forecast Visualization
* Sunrise & Sunset Info
* Rain Probability

---

## 🛠️ Tech Stack

* Power BI
* DAX
* Power Query
* REST API (WeatherAPI)
